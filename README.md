# GAIA_Project

**Un projet complet d'application Flutter AI-powered**

***Le principe ?***

- Gamifier les visites de musées, grâce à une chasse au trésor virtuelle.

- Une oeuvre vous sera présentée, et le but sera de la rechercher dans le musée, pour ensuite la scanner, et la faire rentrer dans sa collection personnelle.

- Attention ! Une fois l'oeuvre scannée, une question sur l'oeuvre sera posée, pour faire assimiler à l'utilisateur des connaissances concrètes sur l'histoire de l'art, et le musée qu'il visite.
